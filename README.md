# eulumdat-ugr

UGR (Unified Glare Rating) calculation from EULUMDAT (.ldt) photometric files — extension to [eulumdat-py](https://pypi.org/project/eulumdat-py/).

> ⚠️ This package is under development. Not yet functional.

## Planned features

- UGR table generation per CIE 117 and CIE 190
- SHR = 0.25 standard spacing
- Room geometry parametrisation (X, Y, H, reflectances)
- Observer position support
